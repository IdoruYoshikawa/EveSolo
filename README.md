# EveSolo
A helper app to gather intel for solo pvp engaments

## For Linux (You might want a venv)
Download EveSolo.py into a folder (i.e. EveSolo) </br>
cd EveSolo </br>
python3 -m venv venv </br>
source venv/bin/activate </br>
pip install requests tkinter </br>
python EveSolo.py

## For Windows
Download the Windows build zip file, unzip it, and execute EveSolo.exe
